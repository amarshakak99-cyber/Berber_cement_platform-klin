# Berber Cement - AI-Powered Kiln Fuel Optimization Platform

## Overview
AI-powered platform for optimizing kiln fuel consumption and temperature control in cement manufacturing.

## Features
- 🔥 Real-time temperature prediction using Random Forest AI
- 💰 Dynamic fuel mix optimization (Coal, Tyre Chips, Alternative Fuels)
- 🌿 Cost & CO₂ savings calculation
- 📊 Real-time sensor data simulation
- 👥 Multi-user role-based access control
